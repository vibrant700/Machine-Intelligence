## 方法一:
### 1. 安装依赖

cd problem_2

pip install -r requirements.txt

### 2. 启动后端服务器

cd problem_2/backend

python解释器路径 server.py

服务器将在 http://localhost:5000 启动。

### 3.打开前端页面
cd problem_2/frontend

python解释器路径  -m http.server 8080
### 4.访问网址
http://localhost:8080
