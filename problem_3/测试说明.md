# 测试指南

## 1. 安装依赖
```bash
# cd problem_3
npm install

# 后端
cd backend
npm install
```

## 2. 配置环境变量

```bash
cd backend
cp .env.example .env
```

编辑 `backend/.env`，填写以下必要配置：

```env
# 填：你的某个Python 路径

# 填：通义千问 API 密钥（https://dashscope.console.aliyun.com/）
QWEN_API_KEY=sk-xxxxx

# 填：阿里云 OSS 配置（用于音频文件上传，语音识别需要）
OSS_REGION=oss-cn-beijing
OSS_BUCKET=your-bucket
OSS_ACCESS_KEY_ID=your-key-id
OSS_ACCESS_KEY_SECRET=your-key-secret
```

## 3. 安装 Python 依赖

```bash
pip install yt-dlp
```

## 4. 启动后端

```bash
cd backend
node server.js
```

正常输出：

```
[Server] http://localhost:8080
[Auth] admin/admin 登录
```

## 5. 构建扩展

```bash
cd problem_3
npm run build
```
会在 `extension/dist/` 生成 `main.js` 和 `visionmark.css`。

## 6. 加载 Chrome 扩展

1. 打开 Chrome，访问 `chrome://extensions/`
2. 开启右上角 **扩展程序**里的**管理扩展程序**
3. 点击 **加载未打包的扩展程序**
4. 选择项目的 `extension` 文件夹

## 7. 功能测试

### 7.1 登录测试

- 点击浏览器右上角的扩展图标（VisionMark）
-注册登录

### 7.2 AI 分析测试

1. 打开一个 B站视频页面，例如 `https://www.bilibili.com/video/BV11CFszFE7j`
2. 等待几秒，视频左上角应出现粉色圆形 **AI** 按钮
3. 点击 AI 按钮，右侧滑出侧边栏
4. 侧边栏应依次显示：
   - Loading 骨架屏
   - 视频总结（AI 生成的摘要）
   - 知识点列表
   - 热词标签
   - 时间线片段（高能/跳过）


### 7.3 知识弹幕测试

- 播放到知识点或热词对应时间点时，视频上方出现滚动弹幕
- 知识点弹幕为蓝色，热词弹幕为粉色




## 项目结构

```
problem_3/
├── backend/              # 后端服务
│   ├── server.js         # 入口
│   ├── config.js         # 配置常量
│   ├── .env              # 环境变量（不提交）
│   ├── .env.example      # 环境变量模板
│   ├── auth/             # 认证模块（JWT + SQLite）
│   │   ├── index.js
│   │   └── database/
│   └── audio/            # 音频分析模块
│       ├── routes.js     # API 路由
│       ├── analyzer.js   # 核心业务逻辑
│       └── downloads/    # 音频缓存目录
├── extension/            # Chrome 扩展
│   ├── manifest.json     # 扩展清单
│   ├── content/          # 内容脚本（注入B站页面）
│   │   ├── main.js       # 主逻辑（分析、跳过、弹幕）
│   │   ├── player.js     # 播放器控制
│   │   ├── config.local.js
│   │   └── constants.js
│   ├── popup/            # 弹出窗口（登录界面）
│   ├── sidebar/          # Vue 3 侧边栏组件
│   │   ├── index.js
│   │   └── components/
│   ├── vite.config.js    # 构建配置
│   └── dist/             # 构建产物（npm run build 生成）
└── package.json          # 根目录构建脚本
```
